
## Model Environment
The network is written using Tensorflow 1.4 in Python 2.7. You can install the dependencies by running:
```bash
pip install -r requirements.txt
```

## Model Implementation
To train: run
```bash
python train.py
```
or
```bash
python train.py -h
```

To evaluate, run
```bash
python eval.py
```
or
```bash
python eval.py -h
```
